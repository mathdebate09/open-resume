## Getting Started

```
$ git clone git@github.com:mathdebate09/openn-resume.git
$ cd openn-resume
$ npm i
$ npm run dev
```

## Learning Outcomes
- Leaned on how to use hook `useState()` and various methods for adding keys, lifting states and overall introduction to writing Js the React way!

## Getting Started

```bash
$ git clone git@github.com:mathdebate09/openn-resume.git
$ cd openn-resume
$ npm i
$ npm run dev
```

## Learning Outcomes

- Learned on how to use hook `useState()` and various methods for adding keys, lifting states and overall introduction to writing Js the React way!
